How to Create a New Repository on GitHub

Step 1: Sign up for a GitHub account

    If you don't have a GitHub account, go to github.com and click on "Sign up".

Step 2: Log in to your GitHub account

    Visit the GitHub login page and enter your credentials.

Step 3: Create a new repository

  1. Click the "+" icon in the top-right corner of the page.
  2. Select "New repository" from the dropdown menu.

Step 4: Set up your repository

  1. Choose a name for your repository.
  2. Optionally, add a description.
  3. Choose whether the repository should be public or private.
  4. Check the box to initialize the repository with a README file.
  5. Click the "Create repository" button.

Step 5: Your repository is ready!

Congratulations! You've successfully created a new repository on GitHub.

Step 6: Add files to your repository

Option 1: Creating a new file

Click on "creating a new file" and paste your HTML code or write your code directly in the new file.

Option 2: Uploading an existing file

Click on "uploading an existing file" to open the upload window. Select your HTML file from your computer and click on the "Commit changes" button.

Step 7: Enable GitHub pages

  1. Go to your GitHub repository.
  2. Click on the “Settings” tab.
  3. Scroll down to the “GitHub Pages” section.
  4. Under “Source,” select “main” or your preferred branch.
  5. Click on “Save”.

Step 8:Access Your Deployed Website

  1. After enabling GitHub Pages, a green success message will appear with a link to your deployed website (e.g., https://your-username.github.io/my-static-website).
  2. Click on the provided link to access your live static website.

Congratulations! You have successfully built and deployed a static website using GitHub Pages.
